Requisitos obligatorios para el código generado:
- Leer el token desde Replit Secrets: os.getenv("EFIPAY_API_TOKEN") en Python o process.env.EFIPAY_API_TOKEN en Node.js

       "amount": monto (número, ej 15000),
       "currency": "CLP",
       "description": "Prueba pago desde Replit",

     }
  2. Del response, extraer la payment_url, checkout_url o redirect_url (la que sea el link de pago)
  3. Retornar al frontend: { success: true, url: "esa_url_de_efipay" }
